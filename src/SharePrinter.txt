Windows:
1. Windows + R.
2. enter "\\192.168.130.140".
3. Drag the folder "SharePrinter" to your desktop.

Linux:
1. Open terminal to get your "uid" and "gid" by
	$ id
2. Go to Desktop, create a folder to mount "SharePrinter" folder.
	$ cd ~/Desktop; mkdir SharePrint
3. Install cifs-utils.
	$ sudo apt install cifs-utils
3. Mount "SharePrinter" folder, follow the instructions below.
	1) For me, gid=1000, uid=1000, but yours may be not!
	2) $ sudo mount -t cifs -o username=Guest,gid="your gid",uid="your uid" //192.168.130.140/SharePrinter ./SharePrint
	3) No password, so just press enter.
